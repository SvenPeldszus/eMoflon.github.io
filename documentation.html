eMoflon Documentation

Wiki, FAQs, Mailing Lists, and Changelog

We maintatin a wiki on GitHub, which also contains an FAQ site, where we collect answers to typical questions of eMoflon users.

Changes are documented in a the text file changelog.txt that is shipped with every eMoflon release.

We have set up a Google group to spread news about exciting eMoflon-related information (e.g., new releases).
