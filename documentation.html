eMoflon Documentation

Wiki, FAQs, Mailing Lists, and Changelog

We maintatin a wiki on GitHub, which also contains an FAQ site, where we collect answers to typical questions of eMoflon users.

Changes are documented in a the text file changelog.txt that is shipped with every release.

We have set up some mailing lists for eMoflon users and developers. We strongly encourage everyone using eMoflon to at least subscribe to the users mailing list

  • Developers: http://liste.es.e-technik.tu-darmstadt.de/mailman/listinfo/moflon-developers
  • Users: http://liste.es.e-technik.tu-darmstadt.de/mailman/listinfo/moflon-users
